🔗 PeerLink

Sichere P2P Kommunikation - Text, Audio & Video

⚠️ GitHub Pages Version - Für vollständige Funktionen verwende die lokale Version

🚀 So startest du PeerLink:

  1. Lokale Version (Empfohlen):
    Lade das Repository herunter und starte den lokalen Server:
    git clone https://github.com/ViewUnitySystemT/TELCOTELEKOM.git
    cd TELCOTELEKOM
    node server.js
    Dann öffne: http://localhost:8080
  2. Mit externem Signalling-Server:
    Konfiguriere einen öffentlichen WebSocket-Server (z.B. auf Heroku, Railway, oder einem VPS)
    Ändere die Signalling-URL in der App-Konfiguration
  3. Docker-Version:
    docker-compose up
    Verfügbar unter: http://localhost:8080
🏠 Lokale Version starten 📁 GitHub Repository 📖 Dokumentation
