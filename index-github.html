🔗 PeerLink

Sichere P2P Kommunikation - Text, Audio & Video

🚀 Jetzt starten 📁 GitHub Repository

✨ Features

  • 🎯 Eine einzige HTML-Datei - keine Installation nötig
  • 🔒 Ende-zu-Ende verschlüsselt mit WebRTC
  • 📱 PWA-fähig - installierbar als App
  • 🎥 Multi-Modi: Text, Audio und Video
  • 📓 Telefonbuch mit Notizen und Medien
  • 🔧 Vollständig konfigurierbar
  • 🌐 QR-Code für einfache Verbindung

💡 Tipp: Für die beste Erfahrung öffne die Anwendung in einem modernen Browser
